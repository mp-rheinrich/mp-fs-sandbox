# This repository is a playground to test the suitability of GlusterFS/Ceph as Media Storage for big video files.

## How to use this project?

    ## to test ceph
    $ sh/ceph
    $ vagrant up

    ## to test gluster
    $ sh/gluster
    $ vagrant up
